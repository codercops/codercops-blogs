## Introduction

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

This tutorial will help you to create a repository on GitHub.

## Prerequisites

- GitHub account
- Git installed on your local machine

## Steps to Create a Repository on GitHub

1. Login to your GitHub account.
2. Click on the `+` icon on the top right corner of the page.

<a href="https://drive.google.com/uc?export=view&id=1S11mweRTNG4554umicrXkk_F0ntml2MI"><img src="https://drive.google.com/uc?export=view&id=1S11mweRTNG4554umicrXkk_F0ntml2MI" styles={{width: 'auto', height: 'auto'}} title="step 2 image" />
</a>

3. Click on `New repository`.

<a href="https://drive.google.com/uc?export=view&id=1fRM7NP4elLP5z10CKM-tK5kmZ_EpSJ5_"><img src="https://drive.google.com/uc?export=view&id=1fRM7NP4elLP5z10CKM-tK5kmZ_EpSJ5_" styles={{width: 'auto', height: 'auto'}} title="step 3 image" />
</a>

4. Enter the repository name and description.

<a href="https://drive.google.com/uc?export=view&id=1VtD4vIK8yMz0X8Trzb-leBQexXVlMHNu"><img src="https://drive.google.com/uc?export=view&id=1VtD4vIK8yMz0X8Trzb-leBQexXVlMHNu" styles={{width: 'auto', height: 'auto'}} title="step 3 image" />
</a>

5. Select the repository type (Public or Private) and click on the `Initialize this repository with a README` option.

<a href="https://drive.google.com/uc?export=view&id=1qkjhFnIPZqU8m7wIyJAf-MOamFxQZBsX"><img src="https://drive.google.com/uc?export=view&id=1qkjhFnIPZqU8m7wIyJAf-MOamFxQZBsX" styles={{width: 'auto', height: 'auto'}} title="step 3 image" />
</a>

6. Click on the `Create repository` button.

<a href="https://drive.google.com/uc?export=view&id=1WINYMe10tk3RT6iYxVA0TV4mfJNGo_76"><img src="https://drive.google.com/uc?export=view&id=1WINYMe10tk3RT6iYxVA0TV4mfJNGo_76" styles={{width: 'auto', height: 'auto'}} title="step 3 image" />
</a>

7. Copy the repository URL by clicking on the `Code` button and selecting the HTTPS option.

![Step 7 image](https://drive.google.com/uc?id=1GzIRDybxBgGeuAM5A6rwMd-WkJ1Pa-T5)

8. Open the terminal on your local machine.

9. Navigate to the directory where you want to create the repository.

10. Run the following command to create a new repository:

```bash
git init
```

11. Run the following command to add the files to the staging area:

```bash
git add .
```

12. Run the following command to commit the changes:

```bash
git commit -m "Initial commit"
```

13. Run the command to change the default branch name from `master` to `main`.

```bash
git branch -M main
```

14. Run the following command to add the remote repository URL you copied in step 7:

```bash
git remote add origin `<repository-url>`
```

15. Run the following command to push the changes to the remote repository:

```bash
git push -u origin main
```
